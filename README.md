# Qt Widgets Projects

A personal learning repository for mastering **Qt Widgets** and modern **C++ desktop application development**.

This repository contains my Qt Widgets learning journey, covering core concepts required for building desktop applications using C++ and Qt.

The goal is to gain hands-on experience with:

* UI Design with Qt Designer
* File Handling
* Dialogs and Menus
* Layout Management
* Model/View Architecture
* Multithreading
* Networking
* OpenCV Integration
